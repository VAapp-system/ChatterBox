# ChatterBox

## Requirement
- System
    - OS: Linux
    - Actions on Google simulator using Dialogflow (not Actions Builder)
- Tool
    - Python3
    - In English
        - Stanford CoreNLP server
        - NLTK
        - Stanza
    - In Japanese
        - JUMAN++ 2.0.0-rc3
        - KNP 4.20
        - pyknp
